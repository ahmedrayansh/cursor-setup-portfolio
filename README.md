This project documents the exact steps I followed after receiving the assignment email. The goal was to set up Cursor, install the required extensions, connect to GitHub, and document the process along with any issues faced and how I handled them.

After receiving the email instructions, I first went through all the steps carefully to understand what was required.

## Step 2: Installing Cursor IDE

* Downloaded Cursor IDE from the official website
* Installed it on my system
* Logged into Cursor successfully

---

## Step 3: Installing Extensions

### Claude Code

* Installed the Claude Code extension successfully
* Tried logging in, but it required:

  * A paid subscription (Pro/Max), or
  * An API key with billing enabled

#### What I Tried

* Attempted normal login
* Explored API key option
* Looked for free access methods through documentation and tutorials
* Explored alternatives like OpenRouter and another tool (likely LM Studio)

#### Outcome

* Installation was successful
* Login was not possible due to paid access requirements
* Alternative tools did not provide access to Claude Code itself
* I decided not to use unofficial or unsafe workarounds

---

### Codex

* Installed Codex extension
* Logged in successfully using my existing ChatGPT subscription
* No issues faced

---

## Step 4: GitHub Repository Creation

* Opened GitHub in the browser
* Logged into my account
* Created a new repository specifically for this project

---

## Step 5: Initial Issue with Git Integration

At this stage, I faced a problem:

* Cursor was not able to properly connect with Git
* Repository cloning was not working as expected

### How I Solved It

* Looked up quick tutorials on cloning repositories
* Realized that Git was not installed on my system
* Installed **Git for Windows**
* Restarted Cursor

---

## Step 6: Cloning the Repository

After installing Git:

* Opened Cursor
* Used `Ctrl + Shift + P` → selected **Git: Clone**
* Pasted the repository URL
* Selected a local folder
* Opened the project in Cursor

### Outcome

* Repository successfully cloned
* Project opened locally without issues
