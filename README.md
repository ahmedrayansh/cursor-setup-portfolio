## Repository Cloning Process

### Initial Issue

The cloning option in Cursor was not working at first.

### Resolution

* Checked short tutorials on how cloning works
* Found that Git was not installed on my system
* Installed Git for Windows
* Restarted Cursor

### Cloning Steps

* Opened Cursor
* Used `Ctrl + Shift + P` → selected **Git: Clone**
* Pasted the repository URL
* Selected a local folder
* Opened the project

### Outcome

The repository was successfully cloned and opened in Cursor.
